# Diachronic Expert Review, demo

A live, static demo of the Diachronic Expert Review platform, for evaluating the interface
and its functionality.

**Open it:** https://tekayanidham.github.io/diachronic-review-demo/

Enter any name to begin. A short guided tour starts automatically. You can browse the corpus,
edit entries inline, leave comments, approve/reject/skip with the pinned buttons, and add a
diachronic.

> This is a demo only. It is static (no server), so a reviewer's work autosaves in their own
> browser and can be exported with "Download study file", but is not collected centrally.
> The real study runs locally or on a self-hosted server where every save is written to disk.
